## How to use this in Obsidian
1. Create a vault called `Jarvis`
2. Drop all .md files from this project into it
3. Open Graph View (Cmd+G on Mac, Ctrl+G on Windows)
4. You'll see every concept linked — click any node to open that doc
5. Use the Daily Notes plugin to log each Jarvis session
6. Use Dataview plugin to query your ideas pipeline like a database

## Plugins recommended
- **Dataview** — query your ideas, tasks, logs like a database
- **Templater** — auto-fill daily brief and weekly review templates
- **Canvas** — visual board for mapping AI ideas
- **Graph View** (built in) — see how everything connects
